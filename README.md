# GRIMOIRE GUARDIANS SCRIPTS
* A Passion Project using Unity Game Engine, C#, and Inky
* A 2D Top-Down Pixel-Art Action Role-Playing Game
* NOTE: Only the game scripts are stored, not the whole game project.

# DOCUMENTATION
Manager Scripts: Contains data (as ScriptableObject), global methods, and are singleton that can persists through scenes 
Controller Scripts: Controls GameObject, communicates with Manager Scripts, and can be prefabs
